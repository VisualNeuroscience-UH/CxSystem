Project Information

About this project:

This is the GeNN project ("genn")

This project was registered on SourceForge.net on Apr 13, 2011.

GeNN is a GPU enhanced Neuronal Network simulation environment based on code generation for NVidia CUDA.

GeNN has been supported by NVidia through a hardware donation to Drs. Luc Berthouze and Thomas Nowotny in the framework of the NVidia-Professor partnerships.
